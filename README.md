## 🖥️ What is cosmodrome?

cosmodrome is a terminal app designed to help you run multiple AI programs side by side. It shows all your projects in one place. You can watch what each AI agent is doing without switching windows or tabs. It works smoothly on macOS but you can also use it on Windows with this guide.

## 🚦 Starting cosmodrome and Running AI Agents

Once cosmodrome is open, here is how to set up and use it:

1. **Create a New Project Tab**  
   Use the “New Project” button inside cosmodrome. Name your project clearly.

2. **Add Your AI Agents**  
   Add instances of the AI tools you want to run, like Claude Code or Codex. You can add them one by one.

3. **Run Agents**  
   Start each AI agent by clicking “Run” next to it. The app will show the live activity for each agent.

4. **View Status**  
   You will see what each AI is doing on the screen. This saves time switching windows or guesswork.

5. **Switch Between Projects**  
   Click to move between different projects to see their agents.

6. **Stop Agents**  
   Use the “Stop” button to end any running AI.

## 🛠 Troubleshooting Tips

- If cosmodrome does not open after installation, restart your computer and try again.  
- Make sure Windows is up to date to avoid compatibility issues.  
- If an AI agent does not start, check your internet connection. Some agents need it.  
- For performance problems, try closing other applications to free up memory.  
- If the app freezes, use Task Manager to close it and reopen.
